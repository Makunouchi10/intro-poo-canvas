# Introducción a la Programación Orientada a Objetos con Canvas 2D

Este proyecto es una introducción práctica a los conceptos de Programación Orientada a Objetos (POO) utilizando HTML5 Canvas y JavaScript.

## Descripción

La aplicación demuestra los fundamentos de la POO mediante la creación de objetos gráficos 2D (círculos) en un elemento Canvas. El proyecto contiene tres demostraciones que ilustran diferentes aspectos de la programación orientada a objetos.

## Características

- **Clase Circle**: Implementación de una clase que representa objetos circulares con propiedades como posición, radio, colores y texto.
- **Objeto 2D Básico**: Demostración de un círculo centrado en el canvas con propiedades fijas.
- **Objeto con Posición Random**: Círculo con coordenadas y radio generados aleatoriamente.
- **Múltiples Objetos**: Creación de un array de 20 círculos con propiedades aleatorias.

## Tecnologías Utilizadas

- **HTML5 Canvas**: API de dibujo 2D para graphics rendering
- **JavaScript (ES6+)**: Programación Orientada a Objetos con clases
- **Bootstrap 5.0**: Framework CSS para diseño responsivo
- **CSS3**: Estilos personalizados

## Instalación y Uso

1. Clona o descarga este repositorio
2. Abre el archivo `index.html` en tu navegador web
3. ¡Explora las tres demostraciones de Canvas!

No se requiere instalación adicional ya que los recursos externos se cargan desde CDN.

## Conceptos POO Demonstrados

- **Clases**: Definición de la clase `Circle`
- **Constructor**: Inicialización de propiedades del objeto
- **Métodos**: Función `draw()` para renderizar el objeto
- **Instanciación**: Creación de objetos mediante `new Circle()`
- **Arrays**: Almacenamiento de múltiples objetos
